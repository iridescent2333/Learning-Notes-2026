#### 第二高的薪水
- select重复子查询，
-     select max(salary) as SecondHighestSalary
      from Employee
      where salary < (select max(salary)from Employee)
      ;
- 使用 IFNULL 和 LIMIT 子句，
- IFNULL，如果表达式结果为空，就返回后面的替代值，没有符合要求的值就返回NULL，IFNULL( 表达式, 当表达式为空时返回的值 )
- distinct，去重，
- ORDER BY A DESC，按A降序，ASC升序，
- LIMIT 1 OFFSET N-1，第N高，LIMIT 1：只取 1 条，OFFSET 1：跳过第 1 条，
-     SELECT
        IFNULL(
          (SELECT DISTINCT Salary
           FROM Employee
           ORDER BY Salary DESC
            LIMIT 1 OFFSET 1),
        NULL) AS SecondHighestSalary
      ;
