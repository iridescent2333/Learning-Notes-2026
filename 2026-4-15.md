#### 查找重复的电子邮箱
- group by a，聚合操作，按列名a将表中其他元素分组，
- count(a)，统计列名a中的元素个数，
- having，用于过滤分组后的结果，可以使用聚合函数(where，过滤原始行，不可使用聚合函数)，
-     SELECT email AS Email
      FROM Person
      GROUP BY email
      HAVING COUNT(email) > 1;
#### 从不订购的客户
-     select c.name as Customers from Customers as c
      left join Orders as o 
      on c.id=o.customerId
      where o.customerId is null;
#### 删除重复的电子邮件
- “我通过子查询按邮箱分组，找出每个邮箱对应的最小 id，然后用 DELETE 删除所有 id 不在这些最小 id 集合中的记录，从而保留每个邮箱中 id 最小的唯一记录。”
- delete，只能删除行，不能接列名，不能直接引用表进行聚合查询，
- min()，函数，寻找列名里面的最小元素，
- 用select多层子查询,as定义子查询结果作为临时列表，
- where加列名加not in(列名)，
-     DELETE FROM Person
      WHERE id NOT IN (
      SELECT min_id
      FROM (
          SELECT MIN(id) AS min_id
          FROM Person
          GROUP BY email
      ) AS t
      );
