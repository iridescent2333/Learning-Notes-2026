#### 查找重复的电子邮箱
- group by a，聚合操作，按列名a将表中其他元素分组，
- count(a)，统计列名a中的元素个数，
- having，用于过滤分组后的结果，可以使用聚合函数(where，过滤原始行，不可使用聚合函数)，
-     SELECT email AS Email
      FROM Person
      GROUP BY email
      HAVING COUNT(email) > 1;

#### 删除重复的电子邮件
- “我通过子查询按邮箱分组，找出每个邮箱对应的最小 id，然后用 DELETE 删除所有 id 不在这些最小 id 集合中的记录，从而保留每个邮箱中 id 最小的唯一记录。”
- delete，只能删除行，不能接列名，不能直接引用表进行聚合查询，
- min()，函数，寻找列名里面的最小元素，
- 用select多层子查询,as定义派生表(derived table)，子查询结果作为临时列表，任何派生表都需要一个别名(Every derived table must have its own alias)，
- where加列名加not in(列名)，
-     DELETE FROM Person
      WHERE id NOT IN (
      SELECT min_id
      FROM (
          SELECT MIN(id) AS min_id
          FROM Person
          GROUP BY email
      ) AS t
      );
#### 上升的温度
- “我使用自连接将 Weather 表连接自身，通过 DATEDIFF函数找到前一天的记录，然后筛选出当天温度高于前一天温度的日期 id。”
- 自连接，把一张表as起别名复制两份，
- datadiff(data1,data2)=1，前者的日期比后者的日期晚一天，日期函数
-     SELECT w1.id
      FROM Weather w1
      JOIN Weather w2
      ON DATEDIFF(w1.recordDate, w2.recordDate) = 1
      WHERE w1.Temperature > w2.Temperature;
