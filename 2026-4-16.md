## 算法设计与分析
### 动态规划
#### 0-1背包问题
##### 问题描述
给定：一个容量为 W的背包,n个物品,第 i个物品：重量：w[i] 价值：v[i]
限制条件：每个物品 只能选一次（要么选，要么不选）总重量不能超过 W
目标：在不超过背包容量的前提下，最大化总价值
##### 存储结构
-     //二维DP
      #include <stdio.h>

      #define MAXN 100
      #define MAXW 1000

      int max(int a, int b) {
          return a > b ? a : b;
      }

      int main() {
          int n, W;
          int w[MAXN], v[MAXN];
          int dp[MAXN][MAXW] = {0};

          // 输入
          printf("请输入物品数量 n 和背包容量 W：");
          scanf("%d %d", &n, &W);

          printf("请输入每个物品的重量和价值：\n");
          for (int i = 1; i <= n; i++) {
              scanf("%d %d", &w[i], &v[i]);
          }
      
          // 动态规划
          for (int i = 1; i <= n; i++) {
              for (int j = 0; j <= W; j++) {
                  if (j < w[i]) {
                      dp[i][j] = dp[i - 1][j];
                  } else {
                      dp[i][j] = max(dp[i - 1][j],
                                     dp[i - 1][j - w[i]] + v[i]);
                    }
              }
          }

          // 输出最大价值
          printf("最大价值为：%d\n", dp[n][W]);

          return 0;
      }
-     //一维DP
      #include <stdio.h>

      #define MAXW 1000

      int max(int a, int b) {
          return a > b ? a : b;
      }

      int main() {
          int n, W;
          int w[100], v[100];
          int dp[MAXW] = {0};

          printf("请输入物品数量 n 和背包容量 W：");
          scanf("%d %d", &n, &W);

          printf("请输入每个物品的重量和价值：\n");
          for (int i = 0; i < n; i++) {
              scanf("%d %d", &w[i], &v[i]);
          }

          // 一维 DP
          for (int i = 0; i < n; i++) {
              for (int j = W; j >= w[i]; j--) {
                  dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
              }
          }

          printf("最大价值为：%d\n", dp[W]);

          return 0;
      }
##### 逻辑结构
- 用动态规划，循环遍历二维数组，如果背包容量不够就沿用前一次的值，如果背包足够就判断并赋值不选物品和选物品的情况结果更大的值，遍历完成后，dp[i][j]即为所求
