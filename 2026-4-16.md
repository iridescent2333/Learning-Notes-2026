#### 订单最多的客户
- where不能用聚合函数，用having来过滤分组后的结果，
- count(*)，列名表示订单数，星号，
- 多层子查询，派生表，( ),
-     SELECT customer_number
      FROM Orders
      GROUP BY customer_number
      HAVING COUNT(*) = (
          SELECT MAX(cnt)
          FROM (
              SELECT COUNT(*) AS cnt
              FROM Orders
              GROUP BY customer_number
          ) AS t
      );
