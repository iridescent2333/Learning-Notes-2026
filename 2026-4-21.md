#### 分数排名
- 同一个表起多个别名，相关子查询
- 同分同名连续排名
-     SELECT
      S1.score,
      (
        SELECT
          COUNT(DISTINCT S2.score)
        FROM
          Scores S2
        WHERE
          S2.score >= S1.score
      ) AS 'rank'
      FROM
      Scores S1
      ORDER BY
      S1.score DESC;
